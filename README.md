# HISPlayer Unity WebGL SDK

HISPlayer is the most advanced Unity video streaming player for WebGL, supporting VOD and live content. It enables premium DASH, HLS video streaming, and WebRTC ultra-low latency inside your Unity web games, immersive 3D websites, and webXR projects across all web browsers on any device. We include advanced features such as multistream, multiview, WebRTC real-time streaming, video ad insertion (SSAI, SGAI & CSAI), subtitles, advanced audio API, and automatic bitrate adaptation to secure the best video quality. You can use the HISPlayer's WebGL Video Player component to control how to time video playback relative to other interactive behaviors in Unity WebGL projects.
